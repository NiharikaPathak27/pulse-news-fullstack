#  PulseNews - React News App

A modern news web application built using React.js that fetches real-time news from News API.

---

##  Features

- Live news fetching using News API  
- Infinite Scroll for continuous news loading  
- Dark Mode toggle  
- Search functionality (Enter + Button)  
- Category filtering (Sports, Technology, India, Politics)  
- Skeleton loading UI  
- Back to Top button  
- Responsive design  

---

##  Tech Stack

- React.js  
- JavaScript  
- CSS3  
- News API  

---

##  How to Run

1. Clone the repository  
2. Open project folder  
3. Run:

```bash
npm install
npm start
